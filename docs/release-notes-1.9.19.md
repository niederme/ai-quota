## What's New in 1.9.19

- **More reliable Claude Code sign-in** — AIQuota now falls back to the web sign-in flow when stored Claude credentials are rejected.
- **Clearer widget gauges** — empty gauge tracks are easier to see while still using system semantic colors.
- **Cleaner widget details** — widgets now match the main app and no longer show a confusing amber `$0.00` spent line.
- **Better macOS 26 readability** — the menu bar popover has stronger contrast when floating over bright windows.
