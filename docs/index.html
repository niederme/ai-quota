FAQ

Questions?

The quick answers on tracking, sign-in, privacy, widgets, and getting set up on your Mac.

What does AIQuota actually track?

AIQuota shows your Codex and Claude Code quota windows, reset timers, and warning states in the menu bar, popover, and widgets. You can see at a glance whether you are in the clear, getting close, or already capped.

Do I need to connect both Codex and Claude Code?

No. You can use AIQuota with Codex only, Claude Code only, or both. If you only connect one service, the app and widgets adapt to that setup instead of leaving empty space for a second one.

How does sign-in work?

AIQuota uses your existing ChatGPT and Claude browser-backed sessions. For normal use, you do not need to paste in API keys. Auth material is stored locally in Keychain, and shared widget data stays in the app group on your Mac.

Does AIQuota read my prompts or code?

AIQuota is built to monitor quota status, not inspect your conversations. It needs account access to fetch usage information, but it does not need your prompt history to show gauges, reset timing, and notifications. You can read more in the Privacy Policy.

Does AIQuota collect usage analytics?

Not by default. AIQuota includes an optional setting called “Share anonymous usage data,” and it starts off. If you turn it on, AIQuota sends simple anonymous metrics: app launch, daily active use, app version, service connection, and onboarding completion. These events may be processed by Google Firebase Analytics. No prompts, tokens, cookies, or personal info.

Do widgets and alerts cover both services?

Yes. AIQuota includes small, medium, and large widget layouts, with single-service and two-service variants where the size allows it. Alerts can let you know when quota is low, critical, exhausted, or reset, so you are not caught off guard mid-session.

What do I need to run it?

AIQuota requires macOS 15 or later. Download the latest notarized build, move it to Applications, launch it from the menu bar, and follow the guided setup to connect your services.
