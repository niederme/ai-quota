## What's New in 1.9.17

- **Smarter Auto refresh** — AIQuota now refreshes every five minutes by default, speeds up to every minute while usage is changing or nearing a limit, and backs off when your Mac is idle, offline, or in Low Power Mode.
- **Clearer budget awareness** — New budget summaries make Codex credits and Claude extra usage easier to understand, with more focused warning states and notifications when Codex credits are replenished.
- **More reliable sign-in** — AIQuota can reuse existing Codex CLI and Claude Code sessions, preserves valid provider sessions across updates, and now correctly hosts Google sign-in popups in its embedded login windows.
- **More resilient usage tracking** — Expanded response handling keeps useful Codex and Claude data visible when providers return newer, partial, or temporarily incomplete usage details.
- **Better on Sequoia** — Popovers, widgets, and onboarding now maintain stronger contrast and more consistent backgrounds on macOS 15, while retaining the native appearance of macOS 26.
