## What's New in 1.9.20

- **New Feature: 🧭 Two gauges in the menu bar** — when Codex and Claude Code are both connected, AIQuota shows a paired double gauge by default while keeping the menu bar item as one click target.
- **Claude Code reconnects after updates** — if an app replacement drops the app-side Claude state but existing Claude Code credentials or a Claude.ai web session are still usable, AIQuota restores the session automatically instead of showing a stale Connect button.
- **Cleaner Settings** — Accounts now includes quiet auth health, refresh diagnostics, and copy diagnostics controls without repeating normal "all good" status in bright green.
- **More Mac-like notifications** — Settings and onboarding now share a single notification hierarchy with a master switch, clickable service disclosure rows, and checkbox-level alert choices.
- **Better Tahoe materials** — the popover uses system materials, semantic text, tertiary gauge tracks, and macOS system red/orange/purple status colors for light and dark appearances.
